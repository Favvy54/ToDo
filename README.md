#  To-Do App

A simple and responsive To-Do List web application that allows users to add and delete tasks. Built with vanilla JavaScript, this project focuses on basic DOM manipulation and clean UI design.

---

##  Features

-  Add new tasks  
-  Delete tasks  
-  Responsive design for mobile and desktop  

---

##  Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)

---

##  How It Works

Users can type a task into the input field and click the add button to include it in the task list. Each task is dynamically rendered on the page, and users can remove tasks when they are no longer needed.



##  Purpose

This project was built to practice:
- DOM manipulation  
- Event handling in JavaScript  
- Responsive UI design
